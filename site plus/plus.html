Nosso projeto como adicional, foi realizado uma inteligência artificial em um ambiente 3D chamado Roblox Studio, escrita em uma linguagem de programação conhecida como Luau, uma derivação da linguagem Lua 5.1. Criamos três inteligências artificiais, no qual tem como objetivo realizar atividades simples.

Introdução
Pathfinder

O pathfinding service é um serviço criado e oferecido gratuitamente pelo Roblox Studio com a finalidade de fazer um personagem humanoid (Com características humanas) desviar de obstáculos ao se mover até uma posição definida. O algoritmo de pathfinding é baseado no algoritmo de Dijkstra, que também é um algoritmo de cálculo de caminho, só que, o pathfinding é um método específico para o Roblox.

O “Pega-Pega entre IAs” é uma proposta de jogo entre dois NPCs que possuem pensamento lógico, onde os mesmos terão como objetivo fazer um jogo de Pega-Pega clássico. Este jogo possuí os personagens “Corredor”, é uma I.A projetada para criar e mover-se até posições lógicas, evitar obstáculos pelo uso de Pathfinding e realizar comportamentos humanos. Ele faz isso pelo seu objetivo principal, que é fugir do outro NPC, que é o “Pegador”. O “Pegador” é uma I.A, mas com o propósito de perseguir o Corredor e eliminá-lo. O pegador não necessita de cálculos de posição, então, o mesmo só segue até o Corredor evitando obstáculos.

Parkour com a I.A

O Jogo de NPC parkour trata-se de uma simples pista de obstáculos e interseções, a qual um NPC percorre para chegar no final do percurso. NPC Parkour: O NPC parkour é uma IA que pula obstáculos, anda e segue posições lógicas por meio do Pathfinding combinado com alguns outros métodos Local do “Parkour”: O local onde é realizado é uma pista de obstáculos feita e revisada por Lucas Henrique.
